Introduction

A browser interprets the html, css and javascript files into a website. Html is the foundtion of a website(It is alone sufficient to create a website and same is not true for css and javascript) It's a markup language (like editors used to markup the manuscripts for authors).

There are elements which requires closing tags with opening tags while some are self closing tags. Attribute provide additional information about HTML element( ex- <hr size="3" noshade>). <!DOCTYPE html> : This informs the type of Document to the browser and version 5 html. <meta charset="utf-8"> : This informs the browser that the texts should be encoded using utf-8(standard encoding) encoded system <meta name="description:" content="A great platform for learning Full Stack Development"> : This provides the description of the website HyperText in HTML is because of the an <a> anchor( or hyperlink) tag which allows to jump from a text document to another one.
