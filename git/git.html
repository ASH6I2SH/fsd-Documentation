GIT AND GITHUB

Steps for clonig a project from github to local device and then making changes to it and the same changes being reflect on github

NOTE: while copying a repository to local system a git file is also downloaded in copied project.

1 Proces:

  1. Installing Git Bash on system

  2. Changing system environment variables by adding path of git bin in system variables path

  3. Installing VS code

3 Process

  1. git clone <repository https url>

  2. cd <repository name>

  3. ls <for list files>
    ls -a <for list files including hidden files like git file>

  4. git status <It shows any modified, untrackable, any renamed or any extra commited file in local system>

  5. git add <name of modified or untrackable file>--- < This is used to add the changes made like modified or untrackable new file created

    OR

    git add . <This adds up all the files wheather a modified or untrackable

  6. git commit -m "<any meaningful message>"

  7. git push origin main < Here the 'origin' refers to the remote repository which is copied>
