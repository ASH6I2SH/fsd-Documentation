Word vs keyword

1. Word is a word which does not has any meaning in javascript. ex- chacha, ashish
2. keyword is a word which has any meaning in javascript. ex- for, if, else, let

var const let

1. var is used for those data storing containers in which data can be changed.
2. const is used for those data storing containers in which data can not be changed.
3. let similar to var.

hoisting

1. hoisting- variable and functions are hoisted which means there declaration is moved on the top of code.
2. If the variable is used before its initilisation the program gives an undefined(value exits but we dont know about it) value while in other progrmming languages it gives not defined(when such value doesn't exit) error.

if else, else if
